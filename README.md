## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up Firebase:
   - Create a Firebase project at https://console.firebase.google.com
   - Copy your Firebase config to `src/config/firebase.ts`

3. Run development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
